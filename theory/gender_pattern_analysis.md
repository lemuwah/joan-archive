# Gender Pattern Analysis — "Her Marke" vs. "His Marke"

Tracks gender-accurate mark conventions across the Fones Record and related documents, forming the linguistic foundation for evaluating the Tier 2 hypothesis (Joan = Anashuecot).

## 1. Core Linguistic Principle

Narragansett has **no grammatical gender**.  
Therefore, any English-language gender marking ("his marke" / "her marke") is:
- **Not** a translation of an Indigenous linguistic convention  
- **Always** a scribe-added English interpretive layer  
- **Potentially** meaningful when consistent  
- **Potentially** misleading when inconsistent  

This principle is the backbone of the gender-pattern lane.
