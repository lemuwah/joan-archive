# Evidence Classification Rubric

## ✅ PROVEN
**Definition:** Direct primary source found, photographed, transcribed, Box.Folder.Page cited.

**Examples:**
- 1682 deed: Joane's signature visible in SK Land Evidence Vol IV, p10
- 1679 affidavit: John Greene's sworn oath — RI Records Vol 56-57

**Criteria:**
- [ ] Photo 600 DPI + IIIF manifest
- [ ] Direct quote transcribed
- [ ] Box.Folder.Page documented
- [ ] Logic chain traced to Joane

## 🔵 PROBABLE
**Definition:** Strong documentary chain, one inferential link, plausible but unconfirmed.

**Examples:**
- Daniel and John Green land deed (Kingstown) — same John? Same period. Probable.
- james1695.jpg probate — Estate mentions John's heir. Circumstantial.

**Criteria:**
- [ ] Relevant primary source located
- [ ] Logical link explained
- [ ] Alternative interpretations noted
- [ ] NOT claimed as PROVEN

## 🟡 UNVERIFIED
**Definition:** Claim exists, source reference vague or not yet pulled.

**Examples:**
- "Awawamicks name variant" — mentioned in Fones but variant spelling not yet located
- "Bristol death absolom kin" — Bristol probate index searched, full record not retrieved

**Criteria:**
- [ ] Source cited but not retrieved
- [ ] Why it matters explained
- [ ] Next step documented

## 🔴 NULL
**Definition:** Searched and not found OR proven compilation layer (WikiTree, Ancestry, Findagrave).

**Examples:**
- La Mance Law — proves nothing about Joane
- alfredgibbs.com map — copied Ancestry without primary anchor

**Criteria:**
- [ ] Search strategy documented
- [ ] Archives/databases searched
- [ ] Negative evidence is data
- [ ] Kept in `/contamination/` not deleted
