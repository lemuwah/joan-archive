# How to Contribute to the Joane Greene Archive

## 🚀 Quick Start

### I want to add a primary source
1. See [`RESEARCH_QUEUE.md`](./RESEARCH_QUEUE.md) — pick a PENDING task
2. Scan at 600 DPI (see [`/digitization/equipment.md`](./digitization/equipment.md))
3. Upload to `/images/`
4. Add entry to `MANIFEST.md` — use [`/digitization/RUBRIC.md`](./digitization/RUBRIC.md) to classify
5. Open a PR

### I want to verify evidence
1. See [`/digitization/RUBRIC.md`](./digitization/RUBRIC.md) — understand classification
2. Pick a PROBABLE or UNVERIFIED entry from [`MANIFEST.md`](./MANIFEST.md)
3. Archive research workflow: Box.Folder.Page required
4. Update MANIFEST.md + add direct quote

### I want to analyze kinship
1. See [`/theory/`](./theory/) for kinship network format
2. See [`/contamination/`](./contamination/) for what NOT to trust
3. Follow Four Laws in [`README.md`](./README.md)

---

## 📋 Submission Checklist

- [ ] Image is 600 DPI (HERO) or 300 DPI (supporting)
- [ ] File named: `[YEAR]_[source]_[page]_[logic].png`
- [ ] MANIFEST.md row added with:
  - Filename
  - Type (1672 Deed, Gender Comparator, etc.)
  - Logic (why this helps Joane)
  - Reference Note (Box.Folder.Page)
  - Status (use RUBRIC.md to classify)
  - Tier (1/2/3/4 or unclassified)
- [ ] Direct quote transcribed in MANIFEST
- [ ] GitHub Actions passes validation

---

## ❌ What We Don't Accept

- Images without primary source anchor
- Claims citing WikiTree/Findagrave without primary
- Speculation presented as PROVEN
- Images deleted or MANIFEST rows removed
